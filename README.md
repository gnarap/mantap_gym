## Teknologi

- HTML5
- CSS3 (Flexbox & Grid)
- JavaScript Vanilla

Tidak menggunakan framework apapun (no Bootstrap, no Tailwind, no jQuery).

## Cara Menjalankan

Download semua file, lalu buka `index.html` di browser. Tidak perlu instalasi apapun.
